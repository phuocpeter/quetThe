QUÉT THẺ
========

Dự án khoá cửa bằng thẻ.

Lắp Mạch
--------

- Yêu cầu:
 - Arduino Board
 - Data Logging Shield
 - Nguồn 5V
 - 3 Đèn LED
 - Loa
 - Servo
 - 3 Nút
 - RFID RC522
 - LCD 1602 + I2C

![alt text](https://github.com/phuocpeter19/quetThe/blob/master/mach.png?raw=true "Hướng Dẫn")

--------
